◯非SSL通信か、SSL通信かによって、読み込みパスを変更する ーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーーー

（1）下記を読み込ませる
<script src="https://ajaxzip3.googlecode.com/svn/trunk/ajaxzip3/ajaxzip3-https.js" charset="UTF-8"></script>

※非SSLのときは下記を

<script src="http://ajaxzip3.googlecode.com/svn/trunk/ajaxzip3/ajaxzip3.js" charset="UTF-8"></script>

（2）郵便番号のinput要素に をつける
<input type="text" name="zip" class="txt small03" onKeyUp="AjaxZip3.zip2addr(this,'','address1','address2');" id="chkZip">

（3）自動で住所を出力させるselect、inputの、name属性に、さっきつけた引数を入れる。
<select name="address1" class="txt mgt10 mgb10">
....
<input type="text" name="address2" class="txt medium02">
