（3）formタグに idをつける
<form action="" method="" id="validation">

（4）バリデーションをするform系要素に idをつける
名前→<input type="text" name="" class="txt medium01" id="chkName">

（5）exvalidation.js内で、（3）でつけたidに対し実行する 
$("#validation").exValidation({
	firstValidate: true,
	errInsertPos: 'after',
	errPosition: 'fixed',
	stepValidation: true,
	rules: {
		chkName: "chkrequired",
		chkKana: "chkrequired chkfurigana",
		chkMail: "chkrequired chkhankaku chkemail",
		chkPW: "chkrequired chkhankaku",
		chkZip: "chkzip chknumonly chkrequired",
		chkAddress01: "chkgroup chkselect",
		chkAddress02: "chkrequired",
		chkTel:  "chkrequired chkhankaku chknumonly",
		chkMail01: "chkrequired chkhankaku chkemail",
		chkMail02: "chkrequired chkretype-chkMail01",
		chkCheckbox: "chkcheckbox"
	}
});

（6）（4）で付けたidに対して値を適宜指定する
名前→chkName: "chkrequired"

※最後のプロパティの値の後には「,」はつけない。
