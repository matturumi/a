任意
（1）下記を読み込ませる 
<link rel="stylesheet" type="text/css" href="./files/css/jquery-ui-1.9.2.custom.css" media="tv,screen,print">
<script src="./files/js/jquery-ui-1.9.2.custom.min.js"></script>
<script>
$(function(){
	//datepicker
	var dataPicker = $('#jquery-ui-datepicker');
	if(dataPicker.length){
		$.datepicker.setDefaults($.extend($.datepicker.regional['ja'])); /* 日本語に設定 */
		dataPicker.datepicker({
			showOn: "button",
			buttonImage: "http://alphasis.info/library/images/javascript/jquery/ui/datepicker/jquery-ui-datepicker-buttonimage-24x22.png",
			buttonImageOnly: true,
			dateFormat: 'yy/mm/dd',
			changeMonth: true,
			changeYear: true,
			yearRange: '2014:2015',
			showMonthAfterYear: true
		});
	}
});
</script>

（2）input要素に id="jquery-ui-datepicker"をつける
<input type="text" name="" class="txt medium01" id="jquery-ui-datepicker">

（3）jquery-ui-1.9.2.custom.css内の画像のパスも適宜変更する。
/files/img/jquery-ui-1.9.2.custom/

（4)カレンダーのテーマを変更する場合
下記サイトでダウンロード
http://jqueryui.com/themeroller/
