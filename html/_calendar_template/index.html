■注意事項
月の移動を前後3ヶ月に制限している。無制限にする場合は、
jquery.calendar_月移動3ヶ月制限なし.js
の方を使用。

ファイル読み込み

<link rel="stylesheet" type="text/css" href="./event_calendar.css" media="tv,screen,print">
<script src="./jquery.calendar.js"></script>

html

<div id="calendar"></div>

js

$("#calendar").calendar({
	navi: {
		ja: ['\u524d\u6708', '\u6b21\u6708']
	},
	// イベントの設定
	events: {
'2015-1-1': {
url: '/event/?act=search&term_bgn=2014/01/08&term_end=2014/01/08'
},
'2015-01-08': {
url: '/event/?act=search&term_bgn=2014/01/08&term_end=2014/01/08'
}
	}
});
