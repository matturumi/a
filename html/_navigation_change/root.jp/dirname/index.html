サイドナビゲーション

ページのurlを取得、サブナビゲーションのhrefの値を見て、
一致したものをアクティブに。

ページ内リンクつきにも対応。

※urlが、「.jp」でないと動きません

グローバルナビゲーション

パンくず、２つ目のliのテキストを取得、グローバルの画像のaltを見て、
一致するものをアクティブに。

※パンくずと、画像のaltに間違いがなく、完全に一致しなくては動きません

/*----------------------------------------------
*
* 	globalNav and side navi active
*
----------------------------------------------*/
(function($){
	$.fn.navActiveChange = function(){
		var self = $(this);

		//urlからside navi active
		var url = document.location.href.split('.jp').pop();
		if(url.indexOf('#')>0){
			url = url.split('#')[0];
		}
		$('#sub a[href="'+url+'"]').addClass('active');

		//topicPathから globalNav active
		var txt = self.find('#topicPath').find('li').eq(1).text(),
				$img = self.find('#globalNav img[alt="'+txt+'"]');
		if($img.size()>0){
			var src = $img.attr('src').replace('_off','_on');
			$img.attr('src',src);
		}
	};
})(jQuery);


$(function(){

	$('#container').navActiveChange();
